# 独立站 05｜新粗野主义 Neo Brutalism

> **推荐指数**：⭐⭐⭐
> **代表**：潮牌 / 设计师个人站 / 艺术工作室
> 
> 

## 适用场景

潮牌、设计师个人站、艺术工作室、Z世代创意产品。

## 设计理念

反精致、反主流柔和审美，粗黑描边、高饱和撞色、错位阴影、像素感排版，张扬个性、年轻叛逆。

## 核心特征

- 2~3px 粗黑实体描边
- 错位悬浮阴影（不模糊、实体投影）
- 高饱和纯色块，无渐变
- 不规则排版、轻微错位
- 无圆角/小圆角

## 色彩

```Plain Text
亮黄：#FFE500
艳蓝：#0066FF
橘红：#FF6B35
纯黑：#000000
纯白：#FFFFFF
```

## 核心 CSS

```css
.brutal-card {
  border: 2px solid #000;
  box-shadow: 6px 6px 0 #000;
  transition: all 0.2s ease;
}

.brutal-card:hover {
  transform: translate(-2px, -2px);
  box-shadow: 8px 8px 0 #000;
}

.brutal-btn {
  border: 2px solid #000;
  box-shadow: 4px 4px 0 #000;
  transition: all 0.2s ease;
}

.brutal-btn:hover {
  transform: translate(-2px, -2px);
  box-shadow: 6px 6px 0 #000;
}

.brutal-yellow { background: #FFE500; }
.brutal-blue { background: #0066FF; color: white; }
.brutal-orange { background: #FF6B35; color: white; }
```

## 组件规范

- **卡片**：实体描边 + 右下错位阴影
- **按钮**：粗边框 + 纯色填充
- **字体**：粗体无衬线，张扬有力

## 优势

极强记忆点、年轻化、小众高级、适配潮流品牌
