# 独立站 08｜暗黑科技赛博风

> **推荐指数**：⭐⭐⭐⭐
> **代表**：Web3 / 区块链 / AI黑科技 / 游戏官网
> 
> 

## 适用场景

Web3、区块链、AI黑科技、游戏官网、极客工具。

## 设计理念

深色基底+霓虹微光+科技线条，未来感、酷感、极客风拉满，低调发光不刺眼。

## 核心特征

- 深黑基底 #0A0A0A
- 青/蓝/紫霓虹微光
- 细科技分割线
- 发光边框hover
- 低饱和发光渐变

## 色彩

```Plain Text
背景：#0A0A0A
主发光：#3CC8C8 / #5B8DEF
文字：#E5E5E5
辅助发光：rgba(60,200,200,0.2)
```

## 核心 CSS

```css
.dark-bg {
  background: #0A0A0A;
  color: #E5E5E5;
}

.cyber-card {
  border: 1px solid rgba(60, 200, 200, 0.15);
  border-radius: 20px;
  background: rgba(30, 30, 30, 0.4);
  transition: all 0.3s ease;
}

.cyber-card:hover {
  border-color: rgba(60, 200, 200, 0.4);
  box-shadow: 0 0 20px rgba(60, 200, 200, 0.15);
}

.text-cyber {
  color: #3CC8C8;
}

.cyber-line {
  height: 1px;
  background: linear-gradient(90deg, transparent, rgba(60, 200, 200, 0.3), transparent);
}
```

## 适配

科技黑科技产品、Web3、元宇宙、游戏、极客工具
